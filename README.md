# Анализатор цветовой палитры

Однофайловое веб-приложение (`index.html`): загрузите референсы — получите общую
цветовую палитру всей подборки. Извлечение цветов — [Color Thief](https://github.com/lokesh/color-thief),
без бэкенда, всё работает в браузере.

**🔗 Живая версия:** https://pronin-artist.github.io/color-palette-analyzer/

## Возможности
- Загрузка нескольких изображений (клик или drag-and-drop)
- Общая палитра по всей подборке (агрегация цветов по частоте)
- Ползунок количества цветов (4–12)
- Клик по цвету — копирование HEX; кнопка «Скопировать все»
- Экспорт палитры в PNG

## Автотесты

Браузерные тесты на [Playwright](https://playwright.dev/python/) + Pillow.

| Файл | Назначение |
|------|-----------|
| `test_full.py` | **Полный автотест: 50 проверок по 15 этапам** — каждый этап использования, его логика, необходимость и качество результата. Сам поднимает локальный HTTP-сервер. |
| `test_app.py` | Базовый smoke-тест локальной сборки |
| `test_live.py` | Проверка опубликованной версии: `python test_live.py <URL>` |
| `make_test_images.py` | Генерация тестовых изображений (включая граничные случаи) |
| `screenshot.py` | Скриншоты пустого и заполненного состояний |

### Запуск полного автотеста
```bash
pip install playwright pillow
python -m playwright install chromium
python make_test_images.py   # сгенерировать тестовые картинки
python test_full.py          # прогнать все 50 проверок
```

### Что покрывает `test_full.py`
0. Загрузка страницы и зависимости (ColorThief)
1. Пустое состояние (скрытие пустых блоков)
2. Фильтрация не-картинок (валидация ввода)
3. Загрузка одного изображения + склонение счётчика
4. Загрузка нескольких изображений
5. Извлечение палитры: валидность HEX, верхний регистр, отсутствие дубликатов
6. Ползунок 4–12: границы и синхронизация подписи
7. Клик по плашке → копирование одного цвета (проверка буфера)
8. «Скопировать все» → формат вывода
9. «Скачать PNG» → валидность и точный размер артефакта
10. Drag-and-drop → визуальная реакция зоны
11. Удаление картинки → пересчёт палитры и счётчика
12. Очистить всё → возврат в пустое состояние
13. Повторное использование после очистки
14. Отсутствие ошибок в консоли за весь прогон
